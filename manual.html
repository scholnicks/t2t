NAME

t2t - text to HTML translator

SYNOPSIS

t2t [Options] Text Files

DESCRIPTION

t2t converts standard ASCII text to HTML tables. Any text with the delimiter embedded in it is converted to a table. The user can specify any regular expression as a delimiter; the default delimiter is the tab. As a result, t2t can easily transform any spreadsheet saved as tab delimited text to a HTML table. The table ends with either the end of the file or the first line of non-delimited text. All other text is just written to the output file.

OPTIONS

--class
CSS class for all tables
--dashes
Skip any line, that starts with 5 dashes or underscores.
--delim [value]
changes delimiter to value
--debug
Turns on debugging mode (implies --verbose)
--equal
Ensures all rows have the same number of columns
--header
Generate a header row (the first row) for each table.
--help
prints out the help screen
--highlight [column number]
Highlight a column header. Must be used in conjunction with header option
--html5
Output in HTML5 mode (--template can override)
--id
CSS id for all tables (only should be used with --one)
--initFilePath [path to initialization file]
Reads in an initialization file from the specified path
--limit [number of columns]
Limits the number of columns. Useful for log files
--nbsp
prints a &nbsp; instead of a blank
--one
Process the entire input file (or all of stdin) as one table. Any line with out a delimiter is written out as a table line with a row with one data module.
--overwrite
controls if t2t will overwrite .html files
--quiet
quiet mode
--squeeze
squeeze out extra fields, i.e. extra delimiters.
--skipfields [value]

Skip field value. Value is a single positive integer.

--style
CSS style for all tables
--tablesOnly
generate only HTML for the tables no surrounding HTML
--template [file]
Use the specified template file
--title [title]
adds a title to the HTML document
--verbose
verbose mode
--version
prints out the current version number
--wholeTable
synonym for --one
